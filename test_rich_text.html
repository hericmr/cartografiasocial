Teste de Rich Text no Painel de Boas-vindas

Instruções

Este teste verifica se o rich text está sendo renderizado corretamente no modal de boas-vindas.

  1. Abra o painel de administração
  2. Vá para a aba "Painel de Boas-vindas"
  3. Use o editor para criar conteúdo com formatação (negrito, itálico, listas, etc.)
  4. Clique em "Ver Preview" para ver como ficará
  5. Salve e verifique se o modal exibe o rich text corretamente
